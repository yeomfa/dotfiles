# My dotfiles in github

## Dependencies

### For commands:

- xclip (Clippboard)
- maim (Screenshots)
- feh (Wallpapers)
- betterlockscreen (Blockscreen)

### For bspwm:

- picom (Transparency and others)
- lxappearance (Themes and more)
- rofi (run, drun, window, launcher)
- polybar (Panel)
- pcmanfm (Files manager)
- dunst (notify)
- sxhkd (commands)

### For terminal:

- terminal (kitty)
- shell (zsh)
- starship (prompt)
